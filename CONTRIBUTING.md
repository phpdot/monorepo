# Contributing

All development happens in this repository. The per-package repositories are read-only mirrors
generated by CI — pull requests and issues there cannot be acted on.

- One package per directory under `packages/`, each with its own `composer.json`, `src/`, `tests/`,
  `LICENSE`, and `README.md`.
- `composer check` must pass before any push: tests, PHPStan (level max, strict), code style.
- To run a single package's tests from the monorepo root:
  `vendor/bin/phpunit -c packages/<name>/phpunit.xml --bootstrap vendor/autoload.php`
- Cross-package changes are one commit here — never coordinate across mirrors.
- Commit authorship is preserved into the mirrors; write commits you want public.
